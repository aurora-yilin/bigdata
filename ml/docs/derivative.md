# 导数

## 方向导数
多元函数的偏导数反映了函数值沿坐标轴方向的变化率，而许多实际问题
中常常还需要掌握函数在某点处沿某一指定方向的变化率。例如，为了预测某
地的风向和风力，必须掌握该地气压沿各个方向的变化率。这就引出了方向导
数的概念。


梯度即函数在某一点最大的方向导数，函数沿梯度方向函数有最大的变化率。
